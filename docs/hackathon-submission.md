# 0GClawForge -- Hackathon Submission

## Track
Track 1: Agentic Infrastructure & OpenClaw Lab

## Summary
0GClawForge is a sovereign agent operating system where teams can mint, orchestrate, own, and evolve multi-agent swarms. Every agent is a tradeable ERC-7857 iNFT on 0G Chain, with encrypted metadata stored on 0G Storage and all inference routed through 0G Compute.

## 0G Component Usage

| Component | Usage | Integration Depth |
|-----------|-------|-------------------|
| 0G Storage | Agent intelligence blobs, persistent memory, cross-session recall | Deep -- core data layer |
| 0G Compute | All AI inference (TEE-verified, micropayment-settled) | Deep -- all inference |
| 0G Chain | iNFT minting, agent registry, marketplace contracts | Deep -- 4 contracts |
| ERC-7857 iNFT | Sovereign agent ownership with secure transfer | Full implementation |
| OpenClaw | 6 native skills + AGENTS.md routing | Full fork + extension |
